# 📊 Data Directory

This folder contains the data structure used in the project.

## 📂 Folder Structure

- `raw/`  
  Original datasets (not included in this repository)

- `panel data/`  
  Cleaned and preprocessed datasets generated from raw data combining all sources

- `extra/`  
  some more segregated datasets

---

## 🔗 Full Dataset Access

Due to size limitations, full datasets are hosted externally:

- Kaggle: https://www.kaggle.com/datasets/tanyatanu1403/terrorism-socio-economic-and-governance-dataset

- Hugging Face: https://huggingface.co/datasets/tanu14/Terrorism

---

## ⚙️ How to Use the Data

1. Download datasets from Kaggle or Hugging Face
2. Place them in the appropriate folders:
   - `data/raw/`
3. Run the notebooks in order:
   - Data cleaning
   - Processing
   - Merging
   - Analysis
